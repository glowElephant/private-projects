# currency-edge

USD/KRW 환차익 자동매매 시스템. 토스증권에서 환전을 자동 반복하여 환차익 수익을 실현합니다.

> **"무릎에서 사고 어깨에서 팔아라"** — 박스권(P15/P85) 기반 계단식 매수 + Claude AI 실시간 전략 최적화

## 시스템 구조

```
┌─ watchdog.py (발터 - 파수꾼)
│   └─ bot/monitor.py — 6슬롯 매매 엔진
│       ├─ 180초 폴링 (±10초 랜덤, 봇 탐지 회피)
│       ├─ 슬롯별 독립 매수/매도 판단
│       └─ 크래시 시 Claude가 자동 수정 → 재시작
│
├─ run_optimizer.py (루도비크 - 길드 참모)
│   ├─ 주간: 10분마다 시장 분석 + 전략 최적화
│   ├─ 야간: 1시간마다
│   └─ 15:30 일일 리뷰 (하루 복기 + 내일 전략)
│
├─ keep_session.py (카스퍼 - 금고실 문지기)
│   └─ 10분마다 Chrome 탭 순환 (세션 유지)
│
├─ discord_bot.py (헤르만 - 전서구 전령관)
│   ├─ 자연어 대화 (환율, 수익률, 슬롯 상태)
│   └─ "!" 명령 → 길드 명령서 → 옵티마이저 실행
│
└─ web/server.py (FastAPI 대시보드)
    ├─ WebSocket 실시간 브로드캐스트 (3분마다)
    ├─ 프로세스 시작/정지, 슬롯 설정 변경
    └─ 로그 뷰어, 설정 관리
```

## 6슬롯 분산 매매

자금을 6등분하여 각 슬롯이 독립적으로 매매합니다.

- **계단식 매수가**: 박스권 무릎(P15) 기준 S1(공격적)~S6(보수적) 자동 배치
- **당일 중복 매수 방지**, 야간/주말 매수 차단
- **마진 하드플로어**: 수수료 미달 시 매도 차단
- **복리 구조**: 매도 후 수익 포함 재매수

## Claude AI 옵티마이저

- **주간 10분 / 야간 1시간마다**: 하나은행 환율 데이터 → 박스권 계산 → 시장 분석 → 전략 최적화
- **일일 리뷰 (15:30)**: 오늘 복기 + 내일 전략 수립
- **장기 기억**: `optimizer_memory.md`에 추세/전략/판단 이력 축적
- **길드 명령서**: 디스코드/웹에서 전략 변경 지시 → 옵티마이저가 최우선 반영

## 자동 복구 (Watchdog)

크래시 감지 → Claude Code가 코드 분석 → 자동 수정 또는 디스코드 알림. Exponential backoff (10→20→40→120초), 최대 3회 시도.

## 웹 대시보드

중세 "환전상 길드" 테마. FastAPI + WebSocket 실시간 모니터링.

| 캐릭터 | 프로세스 | 역할 |
|--------|---------|------|
| 발터 | watchdog | 봇 감시 + 자동 복구 |
| 루도비크 | optimizer | 시장 분석 + 전략 최적화 |
| 카스퍼 | session_keeper | 세션 유지 |
| 헤르만 | discord_bot | 디스코드 챗봇 |

**상인단**: 볼프(S1), 에밀(S2), 한스(S3), 고트프리트(S4), 헬가(S5), 피핀(S6)

## 보안

- RSA 2048비트 암호화 (토스 보안 키패드)
- User-Agent 로테이션 + UUID 생성 (봇 탐지 회피)
- 환경 변수 기반 설정, 세션 만료 자동 감지

## 기술 스택

`Python` `FastAPI` `WebSocket` `Claude Code` `Discord.py` `RSA` `asyncio` `pandas`
